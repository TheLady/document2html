◆ m_codePage

unsigned short excel::Book::m_codePage = 0

An integer denoting the character set used for strings in this file. For BIFF 8 and later, this will be 1200, meaning Unicode; more precisely, UTF_16_LE For earlier versions, this is used to derive the appropriate encoding. Example:

`1252 -> 'cp1252'`, `10000 -> 'mac_roman'`
