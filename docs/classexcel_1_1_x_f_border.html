Detailed Description

A collection of border-related attributes of XF record.

Items correspond to those in the Excel UI's Format -> Cells -> Border tab There are five line style attributes; possible values and associated meanings:

Value Description
0 No line
1 Thin
2 Medium
3 Dashed
4 Dotted
5 Thick
6 Double
7 Hair
8 Medium dashed
9 Thin dash-dotted
10 Medium dash-dotted
11 Thin dash-dot-dotted
12 Medium dash-dot-dotted
13 Slanted medium dash-dotted

Line styles 8 to 13 appear in BIFF8 files (Excel 97 and later) only
